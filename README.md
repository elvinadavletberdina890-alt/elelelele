# Movie Library

**Автор:** Давлетбердина Эльвина
**Язык:** Python 3.x

## Описание

Приложение с графическим интерфейсом (Tkinter) для ведения личной кинотеки. Позволяет добавлять фильмы с проверкой данных (название, жанр, год выпуска — целое число не больше текущего года; рейтинг — от 0 до 10), фильтровать список по жанру и году выпуска.
Все данные сохраняются в файл `movies.json` и автоматически загружаются при запуске.

## Инструкция по запуску

1. Установите Python (версия 3.6+).
2. Установите необходимые библиотеки (если потребуется):
   `pip install -r requirements.txt` *(в данном случае Tkinter встроен в Python)*
3. Запустите приложение:
   `python main.py`
4. Используйте интерфейс для добавления и фильтрации фильмов.
